
# PetrykScript

A mod of YoptaScript (https://github.com/samgozman/YoptaScript)


## Installation

Download PetrykScript.js and place it in the same folder as index.html

    
## Usage/Examples


```html
<!DOCTYPE html>
<html>
    <head></head>
    <body>
        <script language="PetrykScript">
            // Your code here 
        </script>
        <script src="PetrykScript.js"></script>
    </body>
</html>
```
